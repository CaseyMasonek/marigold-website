# Guards

The `guard` keyword is used to prevent excessive nesting in functions. It acts like an early return
for the function, wrapping the following lines in an if statement.

## Usage

`guard ([condition]) [value];`

## Example

<Cell code={`defr factorial(n) {
    # Calculate n! #

    guard (n == 0) 1;

    n * (self n--);
}

putint (factorial 5);`} client:only={"react"} />
